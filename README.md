# 📊 Predictive Analytics Project - Sales Performance Dashboard

## 📌 Project Overview
This project focuses on analyzing historical sales data and creating an interactive Power BI dashboard for predictive analytics and business insights. The dashboard helps identify sales trends, category performance, and key business metrics to support data-driven decision-making.

## 🎯 Project Objective
- Analyze historical sales data
- Identify patterns and trends over time
- Understand category and regional performance
- Track important business KPIs
- Generate actionable insights using data visualization

## 🛠️ Tools & Technologies Used
- Microsoft Power BI
- Power Query
- Excel / CSV Dataset

## 🔄 Data Preparation
The dataset was cleaned and prepared using Power Query:
- Removed duplicate records
- Handled missing values
- Changed data types
- Transformed columns for analysis

## 📈 Dashboard Visualizations

### 🔹 KPI Cards
Created 4 KPI cards to monitor overall business performance:
- Total Sales
- Total Profit
- Total Orders
- Average Sales / Key Metrics

### 🔹 Clustered Bar Chart
Used for category-wise comparison and identifying top-performing segments.

### 🔹 Pie Chart
Used to show contribution/share of different categories in total sales.

### 🔹 Clustered Column Chart
Used to compare performance across categories and analyze differences.

### 🔹 Line Chart
Used to visualize sales trends over time and identify growth or decline patterns.

## 💡 Key Insights
- Identifies sales growth trends
- Highlights profitable and low-performing categories
- Provides a clear view of business performance
- Helps make better strategic decisions using data

## 🚀 Outcome
Built an interactive Power BI predictive analytics dashboard that transforms raw historical data into meaningful visual insights for business improvement.
